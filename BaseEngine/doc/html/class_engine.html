◆ move()

bool Engine::move ( int  fieldNo)
virtual

Służy do poruszania pionkiem gracza. Jako gracz poruszający wybierany jest aktualny gracz.

Parametry
fieldNoPole, z którego ma ruszyć się pionek. Dla wystawienia pionka gracza należy użyć wartości < 0. Aby ruszyć pionek na ostatnich sześciu polach należy wybrać wartości 101-105 201-205 301-305 401-405
Zwraca
true jeśli ruch się powiedzie lub false w przeciwnym razie np. pionek nie stoi na danym polu.
